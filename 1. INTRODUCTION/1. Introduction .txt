2. 크로미움 기반의 웹 브라우저 사용 추천(--> 최신 브라우저) : 
   * 크로미움: 구글 크롬이 베이스로 삼는 프로젝트
     ex ) 네이버 웨일, 브레이브(각종 광고를 차단해주는 브라우저)

3. why javascript?
- 거의 모든 컴퓨터에서 사용
- 웹사이트 만드는 것 이외에도 다양한 곳에 사용 가능
- 자바스크립트는 Netscape Browser에 동적인 요소들을 넣기 위해 새로 만들어진 프로그래밍 언어임
- 유사한 언어들이 있긴 하지만 자바스크립트가 프론트엔드로 쓸 수 있는 유일한 언어
  --> 백엔드와 다르게 여러 가지 언어 중 내가 원하는 것으로 프로그래밍하는 것이 불가능, 오직 자바스크
       립트로만 해야 한다는 장점이자 단점이 될 수 있음. (대체불가)
- 자바스크립트는 HTML과 CSS와 더불어 모든 브라우저에 내장되어 있다. (따로 설치할 필요 X)

4. framework
ex) React Native: 자바스크립트만으로 안드로이드랑 IOS 앱을 만들 수 있게 해줌.
ex) Electron: 자바스크립트, HTML, CSS 만으로 데스크탑 앱을 만들 수 있게 해 줌.

라이브러리 ex) socket.io: 자바스크립트로 실시간으로 무언가를 만들 수 있게 해줌. (도박, 실시간 채팅 등)

이외에도 ml5.js를 이용해 머신러닝 모델을 생성하는 웹사이트를 구축해서 그 모델을 훈련시킬 수도 있다.
