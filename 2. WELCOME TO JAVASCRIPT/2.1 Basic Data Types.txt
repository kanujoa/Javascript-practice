2. console에 집중하기, console에 다음과 같은 내용을 입력하면서 basic data type에 대해 알아본다.
자바스크립트에도 type이 있고 자바스크립트는 기호와 아래와 같은 type들을 인식할 수 있기 때문에 연산이 가능하다.

2-1. number
- interger(full number, 정수)
ex) > 2 + 2
     < 4

- float(실수)      ex) 1.5

2-2. text(정식 명칭은 string, 문자)
- console 창에 그냥 hello라고만 쓰면 오류가 난다. 그러한 값이 정의되지 않았기 때문이다.
string type으로 인식시키기 위해서는 "hello" 처럼 따옴표와 같이 적어주어야 한다.

- string type도 연산이 가능하다.
ex) >"hello" + "my name is nico"
     <"hellomy name is nico"

- 문자열이기 때문에 공백을 주어 작성하는 것도 가능하다.
ex) >"hello " + "my name is nico"
     <"hello my name is nico"
